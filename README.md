# BSP -- 图书共享系统(Book Sharing Platform)

## 简介
该项目是西柚图书共享系统，主要是用户可以将自己的书放在系统上，然后其他用户感兴趣便可以进行 借阅| 收藏 | 预约(当前喜欢的书已经被其他人借走)，分为两套界面，实现功能如下：

- 登陆 | 注册
  - 用户可以注册
  - 管理员不可以注册
- 用户界面
  - 收藏图书
  - 预约图书
  - 借阅图书
  - 修改手机号
  - 修改密码
  - 上传自己的图书
- 管理员界面
  - 审核图书
  - 删除用户
  - 下架书籍
  - 逾期用户查询
